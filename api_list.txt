# A quick descriptions of the APIs

/
GET: returns placeholder page (index)

# DEBUG ONLY
/auctions/
GET: list all auction 

/auction/
POST: starts an auction
    parameters: ...
        itemname
        itemdesc
        itemloc # location
        itemprice
    returns: 
        via the Location: header, the ADMIN UUID
        #a UUID to manage the auction (admin_uuid)
        #a UUID to publish the auction (public_uuid)

/auction/admin_uuid
GET: retrieves auction information + current status of all bids
PUT: updates data
POST: (?) creates a new admin_uuid?

/auction/public_uuid
GET: retrieves the public information
POST: creates a new public_uuid for referrals #TODO
    returns:
        a new UUID bound to the current one (public_uuid)
PUT: performs a bid
    parameters: 
        biddername
        amount  # the bid amount
        question
